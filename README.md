# Hospital Management Dashboard
📖**Project Overview**

This project analyzes hospital operational data to evaluate patient demographics, billing performance, disease trends, and healthcare service delivery using Excel, SQL and Power BI.

The project focuses on transforming raw hospital data into meaningful business insights through data cleaning, KPI development, and interactive dashboards that support operational monitoring and stakeholder decision-making.

📁 **Dataset Overview**

**Source: Hospital Management Dataset**

Structure: Patient, doctor, treatment, billing, and payment records

Key Fields: Patient ID, Doctor, Specialization, Disease, Age, State, City, Treatment, Registration Date, Payment Mode, Bill Amount, Amount Paid, Pending Amount

Goal: Analyze hospital operations, monitor healthcare KPIs, and support business decisions through interactive reporting.

🛠 **Tools & Technologies**

🧹 **Data Preparation**
**a) Data Cleaning**
Removed duplicate and inconsistent records.
Validated patient, billing, and treatment information.
Standardized data formats and handled missing values.
**b) Data Transformation**
Created calculated KPIs for patient, billing, and payment analysis.
Aggregated healthcare metrics using SQL.
Prepared datasets for dashboard development and reporting.

📊 **Business Analysis Performed**
Analyzed patient demographics and age distribution.
Evaluated doctor availability by specialization.
Assessed disease prevalence across patient groups.
Monitored total billing, payments received, and pending payments.
Compared payment modes and state-wise financial performance.
Developed healthcare KPIs for operational reporting.
